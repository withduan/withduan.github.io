恢复浏览器空白页，还你纯净冲浪体验

2022/07/21 chrome newtab sspai

我只想要个纯净的空白页当新标签页…… 怎么就这么难啊？

Matrix 首页推荐 

近日，少数派首页出现了一篇名为《把握浏览器的「门面」：少数派作者们都在用什么启动页？》的文章。在文章中，几位作者分别介绍了自己喜欢的浏览器启动页插件，评论区也有很多读者分享了自己的解决方案。

该文章中多次提到的「启动页」，说的更多的其实是「新标签页」（newtab），而非 Chrome 启动时的页面或主页。在 Chromium 系浏览器中，它的真实路径是 chrome://newtab 。为了避免大家搞混，后面我们统一用「newtab」来称呼它。

可以看出，关于浏览器 newtab 的花样确实很多。网址导航、to-do list、番茄钟…… 各种玩法都很有趣。可我和大家的喜好偏偏相反，我特别钟情于纯空白页，讨厌 newtab 中出现任何元素。🌚

Chrome 浏览器的 newtab 在默认情况下，由早期的空白页（about:blank）不知道什么时候开始变成了 Google 搜索框：

默认 chrome://newtab

如果你想去掉这个搜索框，解决办法是在 Chrome 设置中把地址栏搜索引擎从 Google 改成其他。这时候你会发现，Google 搜索框确实是不见了，可 newtab 的内容又变成了 8 个历史记录中的网站，还关不掉。删掉以后又会出来新的 8 个网站。容易泄露隐私的同时，它给出的建议也不一定用得上，我想要的只有一个 about:blank。

修改搜索引擎后的 chrome://newtab

而且，以 Chrome 浏览器最新版为例，它只能自定义主页和启动页，根本没有提供 newtab 的相关设置。设置主页（chrome://settings/appearance）仅在主动点击「主页」按钮时才会生效；「启动时」设置（chrome://settings/onStartup）仅在浏览器首次启动时生效。这两处设置均无法影响「新标签页」中的内容。

而 Edge 这边的情况更加离谱：

Edge 不知是从哪里得来的设计灵感，newtab 变成了新闻股票大杂烩，用户挣 ~kuī~ 钱吃瓜两不误。可能是担心瓜的热度太高，造成吃瓜群众中暑，还贴心为你展示了天气信息。

默认 edge://newtab

好在 Edge 可以关闭这个 newtab 中大部分的内容，但最后还剩 Bing 搜索框无法去除。左下角还有增值电信业务经营许可证号，真的是想逼死强迫症。🥲

关闭内容展示后的 edge://newtab

安装 Super Evil New Tab Redux

为此我找到了「Super Evil New Tab Redux」这款 Chrome 插件，我的需求才得以解决。这款插件可以为 newtab 设置背景色，可以添加自定义 HTML、CSS、JS 代码。

Super Evil New Tab Redux 选项

插件地址：Chrome 网上应用店 丨 GitHub

插件接管 newtab 之后，默认背景是黑色的，建议修改 Background Color 为白色，或者设置 RGB 值为：255 255 255，这样你就会收获一个纯净的 newtab：

修改源文件

虽然 newtab 变纯净了，但还是有一点点不完美：由于 newtab 被第三方插件接管，因此新建标签页时，标签页左侧 favicon 从  Chrome 图标变成了插件图标，看上去有一些违和。这时我发现插件本身已经开源，可以简单修改一下文件就能达到最理想的效果。

修改newtab.html文件

下载源码并解压，打开 newtab.html文件。

将第 6 行代码中的<title>New Tab</title>修改为：<title>新标签页</title>。

修改插件 icons

去网络中寻找 Chrome 图标，比如在 iconfont 网站中就可以搜到很多这样的图标。

iconfont

选择一个你喜欢的图标，分别下载尺寸为 512、128、96、48、32、16 的 png 文件。打开插件源码中的「icons」文件夹，按照其中的文件命名规律，对刚刚下载的图标文件重新命名，替换原文件。

替换后的 icons

chrome://newtab 默认 favicon 是灰色的，我使用的是彩色 Chrome 图标，你也可以选择原风格的图标。

修改 options.html 文件

顺手对options.html进行简单的汉化，使用起来更加直观。

载入修改版插件

在 Chrome 中打开 chrome://extensions 页面，开启右上方的「开发者模式」。点击「加载已解压的扩展程序」，选取你刚才修改的 Super Evil New Tab Redux 源代码文件夹。此时经过修改后的插件就可以使用了。打开插件设置，修改背景色为纯白色，保存即可。

最终效果

经过修改，插件接管的 newtab 样式已经彻底融入了 Chrome，完全看不出一点违和感。

修改后的 newtab 样式修改后的插件选项

插件同样适用于所有 Chromium 内核浏览器，载入方法也是一样的。

Edge 浏览器使用效果

Edge 浏览器载入该插件后，彩色图标莫名其妙变成了黑白色，但是不影响使用。建议 Edge 用户在上文步骤 2.2 中搜索 Edge 图标进行替换。

Edge：我生气了，哄不好的那种。🙃

你还可以把修改后的插件进行备份，或者放进系统引导盘，让它成为你的装机必备。❤️

其他玩法与方案

自定义 newtab 内容

在上文中我们利用 Super Evil New Tab Redux 获得了全空白的 newtab，但其实 Super Evil New Tab Redux 拥有丰富的自定义功能。除了改变背景颜色，你还可以可以通过编辑 HTML、CSS、JS 代码，定制属于自己的 newtab。

相关推荐

New Tab Redirect

热心派友 @Skyline 和 @UXQZ5C 在评论区中推荐了 New Tab Redirect 这款插件，同样可以自定义 newtab 路径，感兴趣的朋友不妨一试。

插件地址：Chrome 网上应用店 丨 GitHub

cTab

如果你的需求更简单，可以试试热心派友 @loo 亲自动手，制作的一款体积仅有 4KB 的插件「cTab」，插件的唯一功能就是让 newtab 变成空白页。仅仅靠几行代码就实现了需求，真可谓自己动手丰衣足食。😀

插件仅在 Edge 商店中上架，Chrome 用户可以在 GitHub 中下载源码后选择本地载入。

插件地址：Microsoft Edge 加载项 丨 GitHub
